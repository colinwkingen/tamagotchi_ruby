### Colin Kingen 2016
#Technology Used

* Ruby
* Sinatra
* CSS/Bootstrap

## Setup
* Make sure Ruby 2.3.1 is installed.
* Clone repository
* navigate to project directory
* run $bundle install 
* run $ruby app.rb
* navigate to http://localhost:4567
